I am going to make a web application using react as well as a mobile application using flutter and integrate both their information in a database or a cloud server. Also, it will include 2FA authentication and other security features.

Soil Testing Application and AI Prediction for Crop Yielding

The Soil Testing Application works in 3 steps. The first is hardware which involves an Arduino or Raspberry Pi which is attached to a soil sensor to gather data that is sent to the mobile or web application which then uses the data to find out what is missing in the soil by comparing it with the ideal soil amount. This application will be able to tell the amount of Sulphur, Moisture, Phosphorous, and other elements vital for soil to grow good crops.

The AI prediction for the crop yielding will work by using the soil data and weather data along with the crop and land size the user selects to determine the amount of crop-yielding they will have. It will also give them suggestions on which crop will grow the best in the soil and yield more.

Live Car Parking Application

This application will use the sensor (most likely the Weight sensor) to determine if the car is in the parking spot and give the user where is the next parking spot available. I have decided to make the prototype using the IR sensor which will be placed underneath the parking spot which will give data to the microcontroller if the car is in the spot or not an update our application.

This application will allow user to get live updates on the parking location they wish to go to which will allow them to not look for a parking spot after reaching the venue. This will save a lot of time.

Autonomous Trash Bin System

This application will automatically detect the garbage level in the trash bins and determine when they need emptying. This will automatically send an alert to the Waste Management Personnel or Sanitation workers and also let the user know if the bin is full.

This project will use a microcontroller and sensor most effect would be the IR sensor if the bin has a lid on it to protect the sensor from water. If not then we can use the weight sensor to determine the garbage weight and find out the limit of the bin.
